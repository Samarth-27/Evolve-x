# 🚀 PRAGATI-AI - Predictive Real-time Allocation & Governance Assisted Talent Integrator

## 🇮🇳 Revolutionizing the PM Internship Scheme with Advanced AI
**PRAGATI-AI** is a comprehensive, AI-powered internship allocation prototype built for the PM Internship Scheme. It ensures fair, transparent, and ultra-efficient candidate matching while maintaining strict automated quota compliance for affirmative action.

### 🏆 Developed for Smart India Hackathon
> **Live Demo Platform:** Explore the prototype at [evolve-x-ten.vercel.app](https://evolve-x-ten.vercel.app)

## 🧠 Core System Modules

### 🔹 Advanced AI Sandbox Simulation
* **NLP Resume Analysis Engine:** Simulated automated parsing of incoming PDF/DOC uploads to map hidden skills and transferable capabilities.
* **Predictive Success Models:** Real-time generation of student completion forecasting, risk metrics, and structural drop-out factors.
* **Explainable AI UI (XAI):** Clear data-weighting charts showing candidates exactly "Why this match was made" to eliminate algorithmic bias.

### 🔹 Premium Engineering Highlights
* **AI Interview Assistant:** Interactive client-side screening simulator to mock conversational technical evaluations.
* **Skill Gap Mapper:** Visual career path dashboard that compares applicant profiles directly to job demands, identifying target training areas.
* **Gamification Tracking Framework:** Keeps student metrics high with verifiable dynamic talent milestone badges.
